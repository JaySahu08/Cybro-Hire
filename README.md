# Neuro-Hire

Neuro-Hire is a modern recruitment and hiring platform designed to streamline the hiring process for companies and job seekers.

The application provides an interactive and responsive user experience with frontend-backend integration for managing hiring workflows efficiently.

---

## 🚀 Features

- User authentication system
- Recruiter and candidate workflows
- Job posting functionality
- Responsive modern UI
- Frontend and backend integration
- Dynamic data handling
- REST API communication
- Scalable project structure

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/JaySahu08/Neuro-Hire.git
```

---

## 🔧 Backend Setup

### Navigate to backend folder

```bash
cd backend
```

### Install dependencies

```bash
npm install
```

### Start backend server

```bash
npm start
```

---

## 💻 Frontend Setup

### Navigate to frontend folder

```bash
cd frontend
```

### Install dependencies

```bash
npm install
```

### Start frontend

```bash
npm start
```

Frontend runs on:

```text
http://localhost:3000
```

Backend runs on:

```text
http://localhost:5000
```
